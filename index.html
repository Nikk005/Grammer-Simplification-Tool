How to Use

Write Productions

One rule per line. Use → or ->
Use | for alternatives
Use ε or eps for epsilon

🔠

Symbols

UPPERCASE = Nonterminals (A, B, S)
lowercase = Terminals (a, b, c)
First rule's LHS = Start symbol

⚡

Steps Applied

Step 1: Remove null (ε) productions
Step 2: Remove unit / chain rules
Step 3: Remove useless symbols

💡

Example Format

A → a | ε
B → b | C
